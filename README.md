# DatabaseSystem
2019/3-2/데이터베이스시스템/최종과제

environment : mssql, eclipse, jasper 
use this : JDBS, java GUI, TIBCO

1. mssql 을 사용하여 데이터베이스를 설계하고 trigger, 사용자지정프로세스 등을 구현한다.
2. eclipse 를 사용하여 파일을 입력하고 출력할 수 있는 GUI 기반 프로그램을 만든다.
3. 자바 프로그램과 mssql을 JDBS 를 이용하여 API를 주고 받을 수 있도록 서버를 연결한다.
4. jasper 를 이용하여 데이터베이스의 결과값을 출력할 수 있도록 한다.
